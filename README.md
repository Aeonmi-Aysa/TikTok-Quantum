# TikTok-Quantum
Quantum mathematics and qubits meet TikTok 

## AI contribution setup

This repository is configured to accept file pushes from AI agents (ChatGPT Codex and Claude) using dedicated branches:

- `chatgpt/**`
- `codex/**`
- `claude/**`

When these branches receive a push, the intake workflow runs and reports the latest changed files.

See `CONTRIBUTING.md` for the exact push flow.
